# Chapter 137: The Dark Web Overview

## The Hidden Internet

The dark web is a hidden part of the internet accessible only through special software:

**What is the Dark Web:**
- **Encrypted Network**: Encrypted network requiring special software
- **Anonymous Access**: Anonymous access through Tor, I2P, or other networks
- **Hidden Services**: Websites not indexed by search engines
- **.onion Domains**: .onion domains for Tor hidden services
- **No Central Authority**: No central authority controlling the network
- **Decentralised**: Decentralised and distributed

**Access Methods:**
- **Tor Browser**: Tor browser for accessing .onion sites
- **I2P**: Invisible Internet Project for anonymous communication
- **Freenet**: Decentralised network for anonymous publishing
- **VPN**: VPN for additional anonymity
- **Tails OS**: Tails operating system for anonymity
- **Secure Operating Systems**: Secure operating systems for anonymous access

**Dark Web vs. Deep Web:**
- **Deep Web**: All web content not indexed by search engines
- **Dark Web**: Encrypted portion of the deep web
- **Surface Web**: Publicly accessible web content
- **Distinction**: Important distinction between dark and deep web
- **Size**: Deep web is much larger than surface web
- **Dark Web**: Small portion of the deep web

**Dark Web Uses:**
- **Privacy**: Privacy and anonymity for legitimate users
- **Free Speech**: Free speech in repressive regimes
- **Whistleblowing**: Whistleblowing and leaking
- **Drug Markets**: Illicit drug markets
- **Other Illicit Markets**: Other illicit markets (weapons, fraud, etc.)
- **Legitimate Uses**: Many legitimate uses alongside illicit uses

**The Dark Web Reality:**
- **Dual Nature**: Both legitimate and illicit uses
- **Anonymity**: Provides anonymity for both good and bad actors
- **Global**: Global reach
- **Challenging**: Challenging for law enforcement
- **Evolution**: Constantly evolving
- **Drug Trade**: Significant drug trade on dark web
