# Chapter 244: Evaluating Decriminalisation

## Assessing Decriminalisation Outcomes

Evaluating decriminalisation outcomes provides evidence for policy decisions:

**Evaluation Metrics:**
- **Drug Use**: Changes in drug use patterns
- **Overdose Deaths**: Changes in overdose mortality
- **Infectious Disease**: Changes in infectious disease rates
- **Incarceration**: Changes in incarceration rates
- **Treatment Access**: Changes in treatment access
- **Stigma**: Changes in stigma toward drug users

**Portugal Outcomes:**
- **Drug Use**: No significant increase in drug use
- **Overdose Deaths**: Dramatic reduction in overdose deaths
- **HIV/AIDS**: Dramatic reduction in HIV transmission
- **Incarceration**: Significant reduction in incarceration
- **Treatment**: Increased treatment access and engagement
- **Cost Savings**: Cost savings from reduced incarceration

**Other Jurisdiction Outcomes:**
- **European Countries**: Positive outcomes in European jurisdictions
- **US States**: Mixed outcomes from state-level decriminalisation
- **Latin America**: Variable outcomes in Latin American countries
- **Comparative Analysis**: Comparative analysis of outcomes
- **Context Factors**: Context factors affecting outcomes
- **Implementation Quality**: Implementation quality affecting outcomes

**Challenges in Evaluation:**
- **Causality**: Difficulty establishing causality
- **Time Lags**: Time lags between policy and outcomes
- **Confounding Factors**: Confounding factors affecting outcomes
- **Data Quality**: Data quality issues
- **Baseline Comparisons**: Challenges in baseline comparisons
- **Cultural Context**: Cultural context affecting outcomes

**The Evaluation Reality:**
- **Positive Evidence**: Generally positive evidence for decriminalisation
- **Context-Dependent**: Outcomes depend on implementation context
- **Long-Term**: Long-term evaluation needed
- **Evidence-Based**: Strong evidence base for decriminalisation
- **Policy Learning**: Ongoing policy learning
- **Supportive**: Evidence supports decriminalisation as policy option
