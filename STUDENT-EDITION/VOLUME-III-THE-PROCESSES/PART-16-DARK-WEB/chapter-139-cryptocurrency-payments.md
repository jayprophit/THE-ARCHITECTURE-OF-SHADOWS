# Chapter 139: Cryptocurrency Payments

## Digital Currency Transactions

Cryptocurrency is the primary payment method on dark web markets:

**Cryptocurrencies Used:**
- **Bitcoin**: Primary cryptocurrency for dark web transactions
- **Monero**: Privacy-focused cryptocurrency increasingly used
- **Other Cryptocurrencies**: Various other cryptocurrencies used
- **Stablecoins**: Some use of stablecoins
- **Privacy Coins**: Privacy coins for enhanced anonymity
- **Multi-Currency**: Markets accepting multiple cryptocurrencies

**Payment Process:**
- **Wallet Setup**: User sets up cryptocurrency wallet
- **Deposit**: User deposits cryptocurrency into market account
- **Purchase**: User purchases products using cryptocurrency
- **Escrow**: Funds held in escrow until delivery confirmed
- **Release**: Funds released to vendor upon confirmation
- **Withdrawal**: Users can withdraw cryptocurrency

**Anonymity Features:**
- **Pseudonymous**: Transactions are pseudonymous, not anonymous
- **Blockchain Analysis**: Blockchain analysis can trace transactions
- **Mixing Services**: Some use mixing services for additional privacy
- **Privacy Coins**: Privacy coins provide enhanced anonymity
- **Multiple Wallets**: Use of multiple wallets for obfuscation
- **Transaction Patterns**: Transaction patterns can reveal identity

**Law Enforcement Tracking:**
- **Blockchain Analysis**: Law enforcement uses blockchain analysis
- **Exchange Cooperation**: Cryptocurrency exchanges cooperate with law enforcement
- **Seizure**: Seizure of cryptocurrency from markets
- **Tracing**: Tracing cryptocurrency transactions
- **KYC/AML**: KYC/AML regulations on exchanges
- **Investigation**: Investigation of cryptocurrency transactions

**The Cryptocurrency Payment Reality:**
- **Essential**: Essential for dark web markets
- **Not Truly Anonymous**: Not truly anonymous, but pseudonymous
- **Traceable**: Traceable with sufficient resources
- **Evolving**: Constantly evolving to enhance privacy
- **Law Enforcement**: Law enforcement increasingly sophisticated
- **Cat and Mouse**: Constant cat and mouse between markets and enforcement
